# The Rider Residences · Landing Page

A bespoke, conference-ready landing page prepared by Eduba for The Rider Residences (Wynwood Rider, LLC) as part of the eMerge Americas 2026 booth conversation. The page mirrors The Rider's monochrome editorial aesthetic (Didot display serif over a neutral sans, pure black background, thin horizontal rules, zero border-radius, full-bleed whitespace) and delivers the fit-analysis pitch body in the Rider's own rock-and-roll declarative voice. It frames Ciprés Living's operational build (146 short-term rental units, multi-platform distribution, international-buyer onboarding, owner reporting) against Eduba's 60/30/10 orchestration frame and a matching precedent engagement with a 46-property STR operator, then hands off to Matt Creamer (CRO) via a Calendly booking link. All external links open in a new tab with `rel="noopener"`. Mobile responsive down to 375px. Eduba branding is confined to the footer.
